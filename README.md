# About Py-Spread #

Py-Spread是一个Spread Client的纯Python实现.没有任何C的代码实现.

Py-Spread的特性如下：
  * 纯Python实现
  * Spread协议的完整实现
  * 提供完整的单元测试

什么时候使用Py-Spread？
  * 你需要一个纯Python的Spread实现
  * 你需要在Jython中使用他
  * 你需要一个方便调试的客户端

# Install #
